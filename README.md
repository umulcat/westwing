# westwing

This project is completed using WebdriverIO which is a Javascript based test automation framework.
It has 4 JS files.
1. westwinge2e file has e2e test,
including launching the url
accepting cookies alert popup
searching for a product(mobel)
login using the overlay modal form
selecting the first product from the gridlist
and adding it into the wishlist , 
then removing the wishlisted product from iconsbutton tray
2. loginpage file has login code to enter user credentials(can be reused for multiple users)
3. page file has the code to launch the url (can be reused in case of change in the baseurl)
4. search.page file has the code to search the product
